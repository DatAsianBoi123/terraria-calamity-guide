Terraria Calamity Guide Bot

Add to Server View on Github

Features

View Loadouts

You can view any loadout from any class, in any stage of the game by using the command /viewloadout.

Create Playthroughs

You can create playthroughs for a solo or multiplayer playthrough. Create one using /playthrough create, and join onse using /playthrough join. Finally, you can start your playthrough by using /playthrough start

Keep Track of Playthroughs

Terraria Calamity Guide automatically keeps track of your playthrough and the stage of the game you're in. Progress to a different stage of the game using /playthrough progress
